# @sfmc-bds/cli

SFMC 管理 CLI：REPL、服务启停、模块安装、行为包 build/deploy。

## 安装

```bash
npm install -g @sfmc-bds/cli
# 或
npx @sfmc-bds/cli status
```

服主推荐仍使用 [GitHub Releases SEA 单文件](https://github.com/DogeLakeDev/ScriptsForMinecraftServer/releases)。

## 依赖

- `@sfmc-bds/sdk` ^0.1.0
- Node.js >= 22.13
