# @sfmc-bds/cli

SFMC 管理 CLI：REPL、服务启停、模块安装、行为包 build/deploy。

## 安装

```bash
npm install -g @sfmc-bds/cli
# 或
npx @sfmc-bds/cli status
```

服主推荐：

```bash
npm install -g @sfmc-bds/sfmc
```

仅 CLI（不含后端服务）仍可用 `npm install -g @sfmc-bds/cli`，完整平台请装聚合包。

## 依赖

- `@sfmc-bds/sdk` ^0.1.0
- Node.js >= 22.13
