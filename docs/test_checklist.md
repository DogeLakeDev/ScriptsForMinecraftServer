## 二、领地系统 (Land) — !land / !pos1 / !pos2

| # | 操作 | 预期 |
|---|------|------|
| 2.1 | 在空地站好，输入 `!pos1` | 显示已设置第一点 |
| 2.2 | 走到对角，输入 `!pos2` | 显示第二点和预估价格 |
| 2.3 | 输入 `!land` → 点击"确认购买" | 土地创建成功，扣款 |
| 2.4 | 输入 `!land` → 查看土地详情 | 显示你的土地信息 |
| 2.5 | 再次输入 `!pos1` + `!pos2`（和已有土地重叠），然后 `!land` 购买 | 应被拒绝（重叠） |
| 2.6 | 输入 `!land` → 点你的土地 → "成员" → "邀请成员" → 输入另一个玩家的名字 | 邀请成功 |
| 2.7 | 让另一个玩家上线，接受邀请 | 成员加入 |
| 2.8 | 输入 `!land` → 点你的土地 → "转让" → 输入目标玩家名 | 转让成功 |
| 2.9 | **转让后**，让目标玩家输入 `!land` → 点该土地 → "删除" | 删除成功，获得退款 |
| 2.10 | 站到无主之地，输入 `!land here` | 显示"当前位置不在任何土地内" |
