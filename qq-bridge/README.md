# @sfmc-bds/qq-bridge

SFMC QQ ↔ MC 桥接：LLBot OneBot 11 reverse-WS（端口 3002），将群消息转发至 db-server。

## 安装

```bash
npm install @sfmc-bds/qq-bridge
node node_modules/@sfmc-bds/qq-bridge/dist/index.js
```

通常由 `@sfmc-bds/cli` 或 SEA 包统一拉起。

## 依赖

- `@sfmc-bds/sdk` ^0.1.0
- Node.js >= 22.13
