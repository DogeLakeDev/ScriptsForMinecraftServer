# @sfmc-bds/db-server

SFMC SQLite HTTP REST 后端：配置快照、模块启停、游戏数据路由、QQ 消息中继。

## 安装

```bash
npm install @sfmc-bds/db-server
node node_modules/@sfmc-bds/db-server/dist/index.js
```

通常由 `@sfmc-bds/cli` 或 SEA 包统一拉起，无需单独安装。

## 依赖

- `@sfmc-bds/sdk` ^0.1.0
- Node.js >= 22.13（`node:sqlite`）
